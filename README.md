# Personalised Dementia Prediction

## Overview
Personalised Dementia Prediction is a Django-based machine learning web application designed to predict dementia risk using clinical and cognitive assessment data.

## Features
- User Registration & Login
- Dementia Risk Prediction
- Machine Learning Models
- Confusion Matrix Visualization
- Clinical Intelligence Dashboard

## Technologies Used
- Python
- Django
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib
- HTML
- CSS
- JavaScript

## Installation

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
